# Cedar Ops

A dashboard tracking adoption of the Cedar design system — built entirely with Cedar's own components and tokens. Originally prototyped in Lovable, reskinned with Cedar. React 19, TanStack Start, TypeScript, Tailwind v4.

## Stack

- [TanStack Start](https://tanstack.com/start) (file-based routing via TanStack Router)
- React 19 + TypeScript
- Tailwind CSS v4
- bun

## Development

```bash
bun install
bun run dev
```

## Scripts

- `dev` — start the dev server
- `build` — production build
- `lint` — run ESLint
- `format` — run Prettier
